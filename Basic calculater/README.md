# Basic Calculator

## Overview
A fully functional calculator built with HTML, CSS, and JavaScript. Supports:
- Basic operations: +, -, ×, ÷
- Keyboard input
- Order of operations (left-to-right precedence for simplicity)
- Error handling (division by zero)
- Responsive design

## Features
- Click buttons or use keyboard (0-9, ., +, -, *, /, Enter=Equals, Escape/C=Clear)
- Clear (C) resets everything
- Modern, responsive UI

## How to Run
1. Open `index.html` in any modern web browser.
2. Use mouse clicks or keyboard.

**Command:** `start index.html`

## Technologies
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (no libraries)
